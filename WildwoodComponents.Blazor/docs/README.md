# WildwoodComponents Documentation

This folder contains documentation specific to the WildwoodComponents library project.

## Project Overview

WildwoodComponents is a reusable Blazor component library that provides:
- Authentication components (login, registration, OAuth)
- AI configuration components
- Messaging and notification components
- Captcha integration
- Theme and styling services

## Key Areas

- **Components/**: Blazor component usage and integration
- **Models/**: Shared DTOs and data models
- **Services/**: Component services (theme, captcha, etc.)
- **Integration/**: Platform integration patterns
- **Cross-Platform/**: Multi-platform support (Blazor Server, WASM, MAUI)

## Related Documentation

See the main `docs/components/` folder for integration examples and the authentication section for component usage in different contexts.
